# 基于$A^*$的路径规划器

## 项目介绍
该规划器插件使用$A^*$作为主要规划算法，其作为Nav2机器人的一个导航器插件。本插件并没有对$\text{A}^*$规划出来的路径进行额外的线性插值和提前的梯度下降等平滑，也没有进行最大速度最大加速度的限制，这些工作均交给平滑器插件来实现
该插件的名称为`my_nav2_planner/MyAStarPlanner`，可以修改的参数有：
- `unknown_cost` - 未知区域权重
- `interpolation_resolution` - 疏密比率
  
值得注意的是，疏密比率并非必填项，其属于未完成的参数（即暂未实现用途）

## 项目原理
为了正常使用$\text{A}^*$算法，该规划器在导入地图（使用`.pgm`和`.yaml`格式）后，对地图进行划分，采用栅格化地图的方法得到栅格地图，将每一个栅格地图上的位置视作一个节点进行$\text{A}^*$规划，同时，该规划器还应用了对未知区域的考虑，可以自行设置未知权重，以尽可能的让规划的路径原理未知区域
